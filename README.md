# install-mmdvm-sh
Simple script for install mmdvm software on raspberry PI on Debian 9.

Software installing:
* MMDVMHost
  * Wiring PI for LCD display 16Xn
* DMRGateway
* YSFClients
  * YSFGateway
  * YSFParrot
* ircDDBGateway no gui
* DG9VH MMDVMHost-Dashboard
* Telegram BOT
* Button power OFF
* OpenVPN

Read README.txt for exec the script.
